之前有微博朋友留言要我分享一些我做的及我从网上找的一些 Workflow 实例，所以我就在这里整理出来分享给大家，如果大家也有好的 Workflow 实例分享，请点击 [提交](https://github.com/imlifengfeng/WorkflowShare/issues/1)。

【使用方式】直接使用Safari浏览器打开，再点击[GET WORKFLOW]按钮即可添加到自己的Workflow中
